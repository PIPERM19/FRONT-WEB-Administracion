# PMaster frontend

## Herramientas de desarrollo

Creado con:

- vite
- react
- ts
- react router
- tailwind

## Como correr el servidor

```bash
bun i
bun run dev --host
```

o

```bash
npm i
npm run dev -- --host
```

## Notas

El archivo de docker venia con el template de vite y react router, teoricamente puede usarse para el despliegue final pero requiere revision
